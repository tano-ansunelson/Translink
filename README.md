# TransLink - Ghana's Intercity Ride & Cargo Sharing Platform

## Overview
TransLink is a mobile application that connects travelers and truck drivers across Ghana, enabling efficient ride-sharing and cargo transportation between cities.

## User Types
- **Drivers/Truck Owners**: Post upcoming trips for passengers or cargo
- **Passengers/Senders**: Book seats or ship goods using available trips

- **Booking System**: Passengers enter full name, contact, pickup point, and notes

- **Cargo Booking**: Senders provide shipment details (name, contact, locations, load description, weight)
- **Confirmation**: Booking summary with driver information

- **Home Dashboard**: Tabbed interface for Rides/Cargo with quick actions
- **In-App Chat**: Direct messaging between drivers and passengers/senders

- **Notifications**: Real-time alerts for bookings, updates, and trip changes

## Design System
- Modern, minimalist UI with African-inspired colors (green, gold, orange)
- Rounded cards with soft shadows
- Clean typography and intuitive forms
- Icon-based trip/delivery tracking
- Bottom navigation: Home | Search | My Trips | Chat | Profile
